Gives you a display of what is being voted on in Congress in real time.

== Description ==

Somewhat usable now. Moved from alpha to beta. Send feedback to sidney.harrell@gmail.com
Using the Sunlight Labs real-time congress vote tracker API to provide a widget display.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to a widget area
